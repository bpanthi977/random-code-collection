I am trying to wrap my head around back-propagation in a neural network with a Softmax classifier, which uses the Softmax function:

pj=eoj∑keokpj=eoj∑keok

This is used in a loss function of the form

L=−∑jyjlogpj,L=−∑jyjlog⁡pj,

where oo is a vector. I need the derivative of LL with respect to oo. Now if my derivatives are right,

∂pj∂oi=pi(1−pi),i=j∂pj∂oi=pi(1−pi),i=j

and

∂pj∂oi=−pipj,i≠j.∂pj∂oi=−pipj,i≠j.

Using this result we obtain

∂L∂oi===−(yi(1−pi)+∑k≠i−pkyk)piyi−yi+∑k≠ipkyk(∑ipiyi)−yi∂L∂oi=−(yi(1−pi)+∑k≠i−pkyk)=piyi−yi+∑k≠ipkyk=(∑ipiyi)−yi

According to slides I'm using, however, the result should be

∂L∂oi=pi−yi.∂L∂oi=pi−yi.

Can someone please tell me where I'm going wrong?

Your derivatives ∂pj∂oi∂pj∂oi are indeed correct, however there is an error when you differentiate the loss function LL with respect to oioi.

We have the following (where I have highlighted in redred where you have gone wrong)

∂L∂oi=−∑kyk∂logpk∂oi=−∑kyk1pk∂pk∂oi=−yi(1−pi)−∑k≠iyk1pk(−pkpi)=−yi(1−pi)+∑k≠iyk(pi)=−yi+yipi+∑k≠iyk(pi)=pi(∑kyk)−yi=pi−yi∂L∂oi=−∑kyk∂log⁡pk∂oi=−∑kyk1pk∂pk∂oi=−yi(1−pi)−∑k≠iyk1pk(−pkpi)=−yi(1−pi)+∑k≠iyk(pi)=−yi+yipi+∑k≠iyk(pi)=pi(∑kyk)−yi=pi−yi
given that ∑kyk=1∑kyk=1 from the slides (as yy is a vector with only one non-zero element, which is 11).
